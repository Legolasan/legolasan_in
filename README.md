## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

### Build

```bash
npm run build
npm start
```

## Deployment

The site is configured to deploy to a VPS server using Nginx. See the `deploy/` directory for deployment scripts.

## Customization

Edit the data in `src/lib/data.ts` to customize your portfolio content.
